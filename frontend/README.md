# EasyLiving - Modern Real Estate Platform

A premium, responsive real estate web application built with vanilla HTML, CSS, and JavaScript. Find your perfect rental, PG, lease, co-living, or purchase property in one place.

## 🌟 Features

### Core Functionality
- **Property Listings**: 15+ sample properties across 5 categories (Rent, PGs, Lease, Co-living, Buy)
- **Unique Property Images**: Each property has its own distinct high-quality image
- **Advanced Filtering**: Search by location, property type, price range, and keywords
- **Multilingual Support**: English, Hindi, and Kannada language options
- **Responsive Design**: Optimized for desktop, tablet, and mobile devices
- **Separate Category Pages**: Dedicated pages for each property type
- **Interactive Dashboard**: Quick navigation to different property types

### User Experience
- **Smooth Animations**: Modern hover effects and transitions for cards, buttons, and dashboard tiles with **new gradient backgrounds and interactive icons**
- **Alignment Fixes**: Resolved all previous alignment and styling discrepancies.
- **Modal Details**: Detailed property information with contact functionality
- **Sticky Navigation**: Transparent navbar that becomes solid on scroll
- **Premium Design**: Clean, modern interface with gradient backgrounds
- **Accessibility**: Semantic HTML and keyboard navigation support

## 🛠️ Technologies Used

- **HTML5**: Semantic markup and structured content
- **CSS3**: Modern styling with Flexbox, Grid, and CSS Variables
- **Vanilla JavaScript**: No frameworks - pure JavaScript functionality
- **Google Fonts**: Poppins font family for modern typography

## 🚀 Getting Started

### Method 1: Direct File Opening
1. Download all files to the same directory
2. Open `index.html` in any modern web browser
3. The application will load instantly with no dependencies

### Method 2: Local Server (Recommended)
1. Ensure you have Python installed
2. Open terminal/command prompt in the project directory
3. Run: `python -m http.server 8000`
4. Open `http://localhost:8000` in your browser
